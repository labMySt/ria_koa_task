
Skeleton is gotten from [Koajs v2 Application Skeleton](https://github.com/ria-com/node-koajs-skeleton)

quick start
===========

**Using**

```sh
$ cd ria_koa_task
$ npm install
$ npm start

```

**Edit package.json:**

```sh
$ vi package.json
```

**Edit config/* files:**

```sh
$ vi config/default.js
```
